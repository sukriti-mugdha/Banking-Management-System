# Banking-Management-System
This is a Java-based Banking Management System designed using Object-Oriented Programming (OOP) principles. It handles core banking functionalities such as:

- Creating accounts
- Depositing and withdrawing money
- Viewing account details
- Performing transactions securely
- Managing multiple customer records

The system demonstrates use of:
- OOP concepts (Encapsulation, Inheritance, Polymorphism)
- Exception handling
- Collections (`ArrayList`, `HashMap`)
- Menu-driven user interaction via console

> This project is ideal for learning how to structure a simple yet functional application using Java.
